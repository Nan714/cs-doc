**`solver` 是什么？**

逻辑回归内部需要不断调整参数来找到最优解，`solver` 就是"怎么找"的策略：

| solver        | 适合场景        |
| ------------- | ----------- |
| `'lbfgs'`     | 默认值，小数据集，够用 |
| `'sag'`       | 数据量**大**时更快 |
| `'saga'`      | 大数据 + 支持正则化 |
| `'liblinear'` | 小数据集，二分类    |
