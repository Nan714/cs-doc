# And & Or

- **`and`**：找第一个假值，找不到就返回最后一个值
    
- **`or`**：找第一个真值，找不到就返回最后一个值
    
- **返回值类型**：返回的是表达式中**某个操作数的原始值**，不一定会转成布尔值
    
- **只有操作数本身就是布尔值时，才返回布尔值**


# 交互式环境输出
## 快速判断技巧

1. **看到 `print()`** → **一定输出**
    
2. **在交互式环境（>>>）中输入** → **显示返回值**
    
3. **赋值语句** → **不输出**（除非右边有print）
    
4. **函数定义** → **不输出**（除非内部有print）
    
5. **在脚本文件中** → **只有print会输出**
